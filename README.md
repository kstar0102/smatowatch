# 功能说明
已经尽量满足现在智能手表的功能，后期还需要继续添加修改很多东西。大致功能包括：

·日历

·计算器

·秒表

·NFC

·蓝牙传数据给手机、与手机同步时间

·抬腕亮屏

·心率血氧检测（bug还在修复中）

·环境温湿度检测

·指南针

·海拔测量

etc.
