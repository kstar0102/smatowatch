# 功能说明
已经尽量满足现在智能手表的功能，后期还需要继续添加修改很多东西。大致功能包括：

1. 日历
2. 计算器
3. 秒表
4. NFC（刷IC卡）
5. 蓝牙传数据给手机、与手机同步时间
6. 抬腕亮屏
7. 心率血氧检测（bug还在修复中）
8. 环境温湿度检测
9. 指南针
10. 海拔测量
11. 其他功能etc.
