## 增加的功能说明
修改的功能包括：

1. 心率部分，硬件增加LED矩阵，增强PPG信号强度
1. 双板改为贴合，不做4层板因为立创不能做盲埋孔
1. 蓝牙改为KT6328A 
1. MPU6050电路更改，V2.2的AUX接地了，导致之前的功耗高了，现在待机功耗几百uA 
1. 删除NFC部分，因为以前的设计NFC会被PCB铺铜以及屏幕铁皮屏蔽
1. 加入游戏（2048，记忆方块，MPU6050相关游戏）
